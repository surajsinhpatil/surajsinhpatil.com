# surajsinhpatil.com

Personal portfolio website of **Surajsinh Patil** — M.E. Computer Science, BITS Pilani.

Built with [Next.js](https://nextjs.org/) (App Router), [Tailwind CSS](https://tailwindcss.com/), and [Contentlayer](https://www.contentlayer.dev/), and deployed on [Vercel](https://vercel.com/).

## Running locally

```sh-session
git clone https://github.com/surajsinhpatil/surajsinhpatil.com.git
cd surajsinhpatil.com
pnpm install
pnpm dev
```

Create a `.env` file based on [`.env.example`](./.env.example) if you enable the optional pageview counter.
